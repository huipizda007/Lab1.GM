# my_uniq

Консольна утиліта на C#, яка повторює базову функціональність команди uniq.

## Опис

Програма читає рядки з файлу або стандартного вводу (stdin) та видаляє сусідні дублікати рядків.

## Опції

- -i — ігнорувати регістр при порівнянні рядків  
- -c — виводити кількість повторень  

## Приклади

### Вхід:

apple
apple
banana
Banana
cat


### Команда:

my_uniq


### Вивід:

apple
banana
Banana
cat


---

### Команда:

my_uniq -i


### Вивід:

apple
banana
cat


---

### Команда:

my_uniq -c


### Вивід:

2 apple
1 banana
1 Banana
1 cat

## Ввід

- якщо передано файл → читається файл  
- якщо ні → читається stdin  

## Вивід

- результат → stdout  
- помилки → stderr  

## Exit codes

- 0 — успішне виконання  
- 1 — помилка виконання  
- 2 — неправильні аргументи  

## Тестування

Проєкт містить unit-тести (NUnit), які перевіряють:
- правильність роботи програми  
- обробку аргументів  
- обробку помилок  

Тести запускаються автоматично через GitHub Actions при кожному комміті.
